# AI Mock Interview Agent

This project is a simple AI-based mock interview chatbot built using Python.  
It asks questions to the candidate, evaluates their answers using text similarity, and provides feedback and a final score.

## Features
- Asks 5 technical interview questions
- Accepts candidate answers via console
- Evaluates answers using similarity scoring
- Provides feedback: "Good answer" or "Needs improvement"
- Shows final score at the end of the interview

## Technologies Used
- Python
- Basic NLP (text similarity with SequenceMatcher)
